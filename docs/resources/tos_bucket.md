# volcenginecc_tos_bucket (Resource)

存储桶是存储对象（Object）的容器，所有的对象都必须隶属于某个存储桶。存储桶具有各种配置属性，包括地域、访问权限等。您可以根据实际需求，创建不同类型的存储桶来存储不同的数据。

## Example Usage

```terraform
resource "volcenginecc_tos_bucket" "BucketDemo" {
  name                  = "bucket-demo"
  enable_version_status = "Enabled"
  tags = [
    {
      key   = "env"
      value = "test"
    }
  ]
  lifecycle_config = [
    {
      expiration = {
        date = "2027-01-18T00:00:00Z"
      }
      filter = {
        greater_than_include_equal = "Disabled"
        less_than_include_equal    = "Disabled"
        object_size_greater_than   = 123
        object_size_less_than      = 789
      }
      id     = "bucketdemo"
      prefix = "prefix"
      status = "Enabled"
    }
  ]
}
```
