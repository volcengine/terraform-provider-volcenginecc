# volcenginecc_cr_endpoint_acl_policy (Resource)

Adding access addresses to the allowlist for public network access can effectively enhance information security

## Example Usage

```terraform
resource "volcenginecc_cr_endpoint_acl_policy" "CREndpointAclPolicyDemo" {
  registry    = "xxxxx"
  type        = "Public"
  entry       = "0.0.0.0/0"
  description = "test acl policy"
}
```

### Required

- `registry` (String) Name of the image repository instance. On the Instance List page in the Image Repository Console, obtain the instance name
- `type` (String) Network type of the allowlist; currently only Public is supported
