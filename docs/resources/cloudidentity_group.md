# volcenginecc_cloudidentity_group (Resource)

当您需要批量为云身份中心用户进行SSO授权时，您可以先通过用户组进行批量的用户管理。

## Example Usage

```terraform
resource "volcenginecc_cloudidentity_group" "CloudIdentityGroupDemo" {
  description  = "ccapi-multi-des"
  display_name = "test for ccapi"
  group_name   = "ccapi-multi"
  join_type    = "Manual"
  members = [
    {
      user_id = "***********"
    },
    {
      user_id = "*************"
    }
  ]
}
```
