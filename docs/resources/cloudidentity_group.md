# volcenginecc_cloudidentity_group (Resource)

When you need to grant SSO authorization to Cloud Identity Center users in bulk, you can first manage users in bulk through user groups

## Example Usage

```terraform
resource "volcenginecc_cloudidentity_group" "CloudIdentityGroupDemo" {
  description  = "ccapi-multi-des"
  display_name = "test for ccapi"
  group_name   = "ccapi-multi"
  join_type    = "Manual"
  members = [
    {
      user_id = "***********"
    },
    {
      user_id = "*************"
    }
  ]
}
```
