# volcenginecc_cen_grant_instance (Resource)

Authorize this account's network instance for use by the specified account's Cloud Enterprise Network instance

## Example Usage

```terraform
resource "volcenginecc_cen_grant_instance" "CenGranInstanceDemo" {
  cen_id             = "cen-2v72wo******"
  cen_owner_id       = "210102***"
  instance_id        = "vpc-bu3j9gfdv01s****"
  instance_region_id = "cn-beijing"
  instance_type      = "VPC"
}
```
